# weloveburgers

### Description
A Full-Stack Burger Logger application that utilizes a MySQL database and MVC structure.

### Demo
This application is deployed to [Heroku](https://lit-thicket-35467.herokuapp.com/)

### Technologies Used 
- HTML
- CSS
- Handlebars Templating Engine
- Javascript
- Node.js
- Express.js
- MySQL
- MySQL Workbench
- MVC Structure
- Heroku
