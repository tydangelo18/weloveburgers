# weloveburgers

### Description
A Full-Stack Burger Logger application that utilizes a MySQL database and MVC structure.

### Technologies Used 
- HTML
- CSS
- Handlebars Templating Engine
- Javascript
- Node.js
- Express.js
- MySQL
- MySQL Workbench
- MVC Structure
- Heroku
